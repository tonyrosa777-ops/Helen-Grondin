**Key Feature Comparison:**

| Feature | Impact | Traditional Insurance |
|---|---|---|
| Cost split | 90/10 (Impact pays 90%) | 80/20 |
| Deductible structure | Shared family PRA | Per-person deductibles stacked |
| Bill negotiation | 70%+ repricing before member sees the bill | None |
| Rate changes | Stable (locked) | Annual increases |
| Network | No restrictions — any provider | In/out-of-network penalties |
| Contract | Month-to-month | Annual |
| Waiting period | None — coverage starts 1st of next month | Varies |
| ITIN holders | Accepted | Typically excluded |
| Wellness rewards | $150/year credits to lower PRA | None |
| Generic Rx ($10 or less) | $0 | Copay applies |
| Telehealth | 24/7 unlimited, certified doctor | Varies / extra cost |
| Teletherapy | 3 visits/month/topic/member | Rarely bundled |

**Signature Methodology:** The "Impact Model" — community-based health cost-sharing.
Members pool resources to cover each other's medical bills. Bills are repriced 70%+
before the community share kicks in. The PRA (Personal Responsibility Amount) is the
member's only variable cost — shared across the entire household, not per person.
